hidden과 inert 비교 예제

hidden과 inert의 차이를 알아보자

  • hidden은 완전히 요소가 사라집니다. CSS에서 말하는 display:none;과 동일한 효과를 가집니다.
  • inert는 요소는 그대로 있지만 키보드와 스크린 리더를 포함한 접근 수단이 사라집니다.
  • hidden 속성을 주었다 하더라도 해당 예제와 같이 css에서 별도 display 속성을 제공한 경우 css가 우선되어 시각적으로 보일 수 있습니다.
  • Code Pen의 예제는 별도 root.css가 없기 때문에 hidden이 동작하고 있습니다.

hidden이 적용되었지만 root.css로 인하여 display:inline-flex;가 적용되어 요소가 확인됨.
