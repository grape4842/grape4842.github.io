가치관 경매 게임

게임 설명

  1. 당신은 만원 으로 행복하기 위해 필요한 것을 얻는 경매에 참여하게 됩니다.
  2. 최소만원부터 시작하여 금액을 만원 단위로 올려갑니다.
  3. 항목명과 제부 사항을 확인하고 무엇을 얻고 싶은지 생각해 봅시다.
  4. 참여한 다른 사람들과 함께 각자 어떤 결과였는지 이야기를 나눠봅시다.

주의사항

  • 해당 페이지는 단순히 외형만 구현되어 정보가 저장되지 않습니다.
  • 새로고침 할 경우 모든 정보가 초기화됩니다.
