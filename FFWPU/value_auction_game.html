가치관 경매 게임

게임 설명

  1. 여러분께서는 만원 으로 행복하기 위해 필요한 것을 얻는 경매에 참여하게 됩니다.
  2. 최소만원부터 시작하여 금액을 만원 단위로 올려갑니다.
  3. 항목명과 제부 사항을 확인하고 무엇을 얻고 싶은지 생각해 봅시다.
  4. 참여한 다른 사람들과 함께 각자 어떤 결과였는지 이야기를 나눠봅시다.

화폐가 아닌 표를 받았을 경우

  1. 각 품목을 확인하고 얼마를 투자하여 낙찰할지 [예상 사용 금액]란에 적습니다.
  2. 경매를 진행하면서 실제로 사용한 금액을 [실제 사용 금액]란에 적습니다.
  3. 비고란에는 품목을 보며 이 품목을 원했던 이유 등을 적어봅시다.
  4. 경매를 마친 후 예상했던 금액과 실제로 사용한 금액을 비교해봅시다.

사회자가 준비할 내용

  1. 사회자는 총 금액, 최소 금액, 올릴 단위 금액을 결정합니다.
  2. 화폐를 나누어줄지 표를 나누어줄지 선택하여 진행 방식을 선택합니다.
  3. 참가자에게 표 또는 화폐를 나눠줍니다.

❗주의사항

  • 해당 페이지는 단순히 외형만 구현되어 정보가 저장되지 않습니다.
  • 새로고침 할 경우 모든 정보가 초기화됩니다.
