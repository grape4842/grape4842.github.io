제목 설명
표 제목 데이터가 포함된 표는 반드시 표 제목으로 table태그 하위로 caption태그를 제공하여야 한다.
제목 셀과 데이터 셀 데이터가 포함된 표는 반드시 표의 제목셀과 데이터 셀을 태그 단위(th, td)로 나누어야 한다.
제목셀의 방향 제목셀에서 스크린리더가 정보를 주워오는 방향을 가이드해쥐 위해 scope속성을 제공하여야 한다.
중복된 표 테이블 셀 내에 테이블을 마크업 할 경우 스크린리더가 올바르게 인식하기 어려우며 사용자 또한 이해하기 어려우므로 사용해서는 안된다.
복잡한 표에서 제목 셀의 데이터를 가져올 때 복잡한 구조의 표는 제목 셀의 데이터가 다수 필요할 수 있다. 이 떄 제목셀에 id속성을 통해 고유값을 부여하고, 데이터를 가져올 셀에 headers속성을 통해 선언한 id고유값을 불러오는 것으로 스크린리더로 데이터 셀에 접근했을 때 연결된 제목 셀의 내용을 읽어준다.
