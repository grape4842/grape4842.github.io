숨김텍스트 실험

다양한 방법으로 CSS를 활용하여 숨김텍스트를 제공해보고 스크린리더에서 인지가 가능한지, 초점이 튀지는 않는지 테스트합니다. 이 문장은 올바른 숨김텍스트 제공 예시의 CSS로 숨겨져 있습니다. 올바르게 접근되는지 확인해보세요.

  • 1번 예시 : position absolute와 text-indent로 날리는 방법
  • 2번 예시 : overflow:hidden; display:inline-block; position:relative; font-size:1px; z-index:-1; top:-1px; left:-1px; width:1px; height:1px; blind class 적용
  • 3번 예시 : 글자 크기를 0으로 설정할 경우 iOS에서는 탐지할 수 없습니다.
  • 4번 예시 : opacity로 완전히 투명하게 만든 텍스트입니다
  • 5번 예시 : 크기값을 0으로 주면 읽을 수 있을까요
  • 6번 예시 : 크기값을 1만 주면 점은 보일 것입니다
  • 7번 예시 : position으로 날려버린 예시
  • 8번 예시 : z-index 음수를 선언하였지만 덮을 항목이 없어 뒤로 가 있지만 가려지지는 않았습니다. 이 위에 박스를 놓으면 가려지겠죠?
  • 9번 예시 : 글자색을 투명하게 만들어보았습니다.
