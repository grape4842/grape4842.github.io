1. 적절한 대체 텍스트 제공
이미지에 대체텍스트 (alt)속성이 제공되지 않아 스크린리더 사용자는 이미지에 담긴 정보를 인지할 수 없습니다. 이미지의 정보를 인지할 수 있도록 대체텍스트 속성을 제공해야 합니다.
이미지의 대체텍스트 속성값이 실제 이미지의 내용과 다르게 제공되고 있어, 스크린리더 사용자는 잘못된 정보를 인지하게 됩니다. 이미지가 나타내는 정보를 정확하게 인지할 수 있도록 대체텍스트의 속성값을 수정해야 합니다.
이미지의 대체텍스트를 목적으로 제공된 숨겨진 텍스트가 실제 이미지의 내용과 다르게 제공되고 있어, 스크린리더 사용자는 잘못된 정보를 인지하게 됩니다. 이미지가 나타내는 정보를 정확하게 인지할 수 있도록 숨겨진 텍스트를 수정해야 합니다.
장식이나 레이아웃을 위해 배치된 이미지에 대체텍스트 속성값이 제공되고 있어, 스크린리더 사용자는 불필요한 정보를 인지하게 됩니다. 정보를 담고 있지 않고 장식으로 제공되는 이미지는 넘어갈 수 있도록 대체텍스트의 속성값을 공백으로 수정해야 합니다.
장식이나 레이아웃을 위해 배치된 이미지에 (alt)속성이 제공되지 않아 스크린리더 사용자는 불필요한 이미지의 파일명 정보를 인지하게 됩니다. 정보를 담고 있지 않고 장식으로 제공되는 이미지는 넘어갈 수 있도록 대체텍스트 속성을 제공하되, 속성값을 공백으로 제공해야 합니다.
이미지맵을 사용하여 이미지의 내용이 대체텍스트로 충분히 설명되지 않았습니다. 이미지가 나타내는 정보를 정확하게 인지할 수 있도록 대체텍스트의 속성값을 수정해야 합니다.
이미지에 title속성을 통해 대체텍스트가 제공되었으나, 이미지에 대체텍스트 (alt)속성이 제공되지 않아 스크린리더 사용자는 이미지에 담긴 정보를 인지할 수 없습니다. 이미지의 정보를 인지할 수 있도록 대체텍스트 속성을 제공해야 합니다.
QR코드가 제공되었으나 이를 스크린리더 사용자는 이용할 수 없습니다. QR코드의 기능을 동등하게 대체할 수 있는 링크나 기능을 삽입하여야 합니다.
배경으로 제공된 이미지에 대체텍스트가 제공되지 않아 스크린리더 사용자는 이미지에 담긴 정보를 인지할 수 없습니다. 이미지의 정보를 인지할 수 있도록 대체텍스트를 제공해야 합니다.

2. 자막 제공
동영상에 대응하는 자막이 제공되지 않아 일부 사용자는 동영상의 내용을 이해할 수 없습니다. 자막 또는 원고나 수화를 제공하여 모든 사용자가 도영상의 내용을 이해할 수 있도록 제공하여야 합니다.
2동영상에 대응하는 자막이 잘못 제공되어 일부 사용자는 동영상의 내용을 잘못 인지하게 됩니다. 자막 또는 원고나 수화를 올바르게 제공하여 모든 사용자가 동영상의 내용을 정확하게 인지할 수 있도록 수정해야 합니다.
동영상을 제공할 수 있으나 자막, 원고, 수화를 제공할 수단이 제공되지 않았습니다. 동영상과 함께 자막, 원고, 수화를 제공할 수 있도록 기능을 제공하여야 합니다.

3. 색에 무관한 콘텐츠 인식
콘텐츠의 내용의 차이를 색상만으로 구별하였기 때문에 색약사용자는 구별하지 못할 수 있습니다. 색상뿐만이 아닌 다른 방법으로 구별할 수 있도록 수정하여야 합니다.
체크박스의 체크여부를 색상만으로 구별하였기 때문에 색약사용자는 구별하지 못할 수 있습니다. 색상뿐만이 아닌 체크 표시의 유무로 구별할 수 있도록 수정하여야 합니다.

5. 텍스트 콘텐츠의 명도 대비
텍스트의 명도대비가 4.5:1을 넘지 못합니다. 이 떄문에 텍스트와 배경색을 구별하기 어렵습니다. 색상을 수정하여 명도대비를 높여야 합니다. (보통 크기 : 24px 미만, 또는 굵은 19px 미만 텍스트 기준)
텍스트의 명도대비가 3:1을 넘지 못합니다. 이 떄문에 텍스트와 배경색을 구별하기 어렵습니다. 색상을 수정하여 명도대비를 높여야 합니다. (보통 크기 : 24px 이상, 또는 굵은 19px 이상 텍스트 기준)
글자 크기가 19px미만의 이미지 텍스트의 명도대비가 4.5:1을 넘지 못합니다. 이 떄문에 텍스트와 배경색을 구별하기 어렵습니다. 이미지를 수정하여 명도대비를 높여야 합니다.
글자 크기가 19px이상의 이미지 텍스트의 명도대비가 3:1을 넘지 못합니다. 이 떄문에 텍스트와 배경색을 구별하기 어렵습니다. 이미지를 수정하여 명도대비를 높여야 합니다.

20-2. 표의 제목과 요약정보가 불충분하거나
적절하게 제공되지 않은 경우
20-3. 제목 셀과 내용 셀을 구분하지 않거나 잘못 구분한 경우
표의 제목셀에 scope속성이 제공되지 않아 스크린리더 사용자가 표를 탐색할 경우 데이터 셀이 표의 제목 셀의 값을 참조하지 않아 표를 이해하는데 불편할 수 있습니다. Scope속성을 제공하여 표를 탐색하기 쉽도록 수정하여야 합니다.
20-4. 다단, 병합 등 복잡한 표를 제공 시
headers, id 속성을 통해 제목 셀과 내용 셀을 연결하지 않은 경우
20-5. 표의 정보를 이해하기 어려운 중첩 표
(표 안의 표)를 넣어 정보를 제공한 경우

21. 레이블 제공
21-1. <input>, <textarea>, <select> 요소에
1:1 대응하는 <label>요소 또는
title 속성을 제공하지 않거나 설명이 적절하지 않은 경우
label로 제공할 수 있는 텍스트가 제공되었음에도 불구하고 label태그를 통해 연결하지 않고 title속성으로만 입력서식의 목적을 제공하였습니다. label태그를 통해 입력서식과 연결하도록 수정하여야 합니다.
21-3. <input>의 id와 <label>의 for가 다르거나,
페이지 안에 같은 id가 있는 경우

22. (오류 정정)
입력 오류를 정정할 수 있는 방법을 제공해야 한다.
22-1. 입력 오류 발생 시, 오류 원인 및 내용을
적절한 설명 텍스트로 알려주지 않는거나 적절하지 않은 경우
22-2. 입력 오류 발생 시, 정정할 수 있는 수단을 제공하지 않은 경우
22-3. 오류 정보 확인 시,
해당 입력 서식으로 초점이 이동하지 않는 경우
22-4. 입력 오류 발생 시 입력 내용이 모두 사라지는 경우

23-3. 중복 선언된 속성 오류

24. (웹 애플리케이션 접근성 준수)
