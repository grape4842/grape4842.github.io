1. 적절한 대체 텍스트 제공

이미지의 대체 텍스트 속성값이 실제 이미지의 내용과 다르게 제공되어 스크린리더 사용자는 이미지의 내용을 잘못 전달받게 됨. 이미지의 내용을 올바르게 전달받을 수 있도록 대체 텍스트의 속성값을 수정해야 함.

이미지의 대체 텍스트를 숨김텍스트 형태로 제공하였으나 실제 이미지와 달리 많은 내용이 통째로 영역구별 없이 제공되어 내용을 알기 어려움. 실제 이미지와 유사하게 사용자가 내용을 파악하기 쉽도록 태그를 사용하여 영역별로 나눌 것을 권장.
이미지의 대체 텍스트로 제공된 숨김텍스트가 실제 이미지의 내용과 달라 스크린리더 사용자는 이미지의 내용을 잘못 전달받게 됨. 이미지의 내용을 올바르게 알 수 있도록 숨김텍스트를 수정해야 함.

대체텍스트와 동일한 내용이 이미 텍스트로 충분히 제공됨에도 이미지에 동일한 대체텍스트가 제공되어 불필요한 탐색이 발생함. 이미지의 대체 텍스트의 속성값을 공백으로 제공하여 탐색되지 않도록 수정해야 함.
이미지와 동일한 내용이 텍스트로 제공되어 이미지에 대체 텍스트를 제공할 필요가 없음. 불필요한 탐색을 줄이기 위해 대체 텍스트를 공란으로 수정해야 함.
이미지맵을 사용한 이미지의 내용이 대체 텍스트로 충분히 설명되지 않음. 이미지가 나타내는 정보를 정확하게 알 수 있도록 대체 텍스트의 속성값을 수정해야 함.
표를 이미지로 제공하여 표의 내용을 이해할 수 없음. 엑셀파일과 같은 대체수단을 마련하여 스크린리더 사용자도 표의 내용을 이해할 수 있도록 제공해야 함.
이미지에 title속성을 통해 대체 텍스트가 제공되었으나, 이미지에 대체 텍스트 (alt)속성이 제공되지 않음. 이미지에는 title속성으로는 대체 텍스트가 제공되지 않음. 따라서 스크린리더 사용자는 이미지에 담긴 정보를 인지할 수 없음. 이미지의 정보를 인지할 수 있도록 대체 텍스트 속성을 제공해야 함.
QR코드가 제공되었으나 이를 스크린리더 사용자, 키보드 사용자는 이용할 수 없음. 모든 사용자가 QR코드의 기능을 사용할 수 있도록 동등한 기능을 제공해야 함.
배경 이미지에 대체 정보가 제공되지 않아 스크린리더 사용자는 이미지의 내용을 수 없음. 스크린리더 사용자가 이미지의 내용을 알 수 있도록 대체 정보를 제공해야 함.
배경 이미지에 대체 정보가 제공되었으나, 스크린리더가 이를 인지하지 못하기 때문에 사용자는 이미지의 내용을 알 수 없음. 스크린리더 사용자가 이미지의 내용을 알 수 있도록 대체 정보를 제공해야 함.
이미지를 제공할 수 있으나 대체 텍스트를 제공할 수단이 제공되지 않았습니다.
이미지와 함께 대체 텍스트를 제공할 수 있도록 기능을 제공해야 합니다.
첨부파일로 제공된 PDF파일이 이미지로 제공되어 파일의 내용을 스크린리더로 확인할 수 없음.
숨김 텍스트가 스크린리더가 인지할 수 없는 형태로 제공됨. 스크린리더가 인지할 수 있도록 제공방법을 수정해야 함.

2. 자막 제공

영상에 자막이 잘못 제공되어 동영상의 내용을 알 수 없음. 자막 또는 원고와 수화를 수정하여 모든 사용자가 영상의 내용을 알 수 있도록 제공해야 함.

3. 색에 무관한 콘텐츠 인식
콘텐츠의 내용의 차이를 색상만으로 표현했기 때문에 색상을 구별할 수 없는 경우, 콘텐츠의 차이를 알 수 없음. 색상뿐만이 아닌 다른 방법으로 콘텐츠의 차이를 구별할 수 있도록 수정해야 함.
체크박스의 체크여부를 색상만으로 구별하였기 때문에 체크 여부를 알 수 없음. 색상뿐만이 아닌 체크 표시의 유무로 체크 여부를 명확히 알 수 있도록 수정해야 함.

8. 키보드 사용 보장
기능이 제공된 요소가 마우스 / 터치로만 조작할 수 있도록 구현됨. 키보드만으로도 기능을 조작할 수 있도록 마크업을 수정해야 함.
키보드로 접근한 경우 마우스 대비 제공되는 정보가 마우스 대비 부족함. CSS를 수정하여 마우스가 접근한 경우와 동등한 정보가 제공되도록 수정해야 함.
일반 텍스트에 기능이 부여되어 있어 요소의 역할을 예측할 수 없어 조작하는 요소임을 알 수 없음. 요소의 역할을 알 수 있도록, 사용자가 조작할 수 있도록 수정해야 함.
기능이 제공되는 요소에 스크린리더가 접근할 수 없음. 소스를 수정하여 초점이 접근할 수 있도록 수정해야 함.
멀티터치로만 사용 가능한 기능을 한 손가락으로도 사용할 수 있도록 대체 수단을 제공하지 않음. 한 손가락으로도 조작할 수 있는 대체 수단을 제공해야 함.

9. 초점 이동
초점이 예측할 수 없는 형태로 이동함. 초점이 논리적으로 이동하도록 수정해야 함.
요소를 활성화할 경우 초점이 초기화되어 처음부터 재탐색해야 함. 요소를 활성화하여 변화가 생긴 경우 변화를 알 수 있도록 초점이 이동하도록 제공해야 함.
초점이 접근하는 순서가 논리적이지 않아 콘텐츠의 흐름을 파악할 수 없음. 마크업상 요소가 제공되는 순서를 변경하여 초점의 접근 순서를 수정해야 함.
초점이 접근하는 순서가 논리적이지 않아 콘텐츠의 흐름을 파악할 수 없음. CSS를 수정하여 초점의 접근 순서를 수정해야 함.
초점이 특정 영역에 갇혀 더 이상 이동할 수 없음. 초점이 이동할 수 있도록 수정해야 함.
더보기를 활성화 한 경우 초점이 추가된 항목으로 자동적으로 이동하지 않음. 더보기로 콘텐츠가 추가된 경우 추가된 콘텐츠로 자동적으로 이동하도록 수정해야 함.
화면을 덮는 레이어가 팝업되었으나 자동적으로 초점이 레이어로 이동하지 못해 현재 레이어가 팝업됨을 알기 어려움. 레이어가 팝업되면 자동적으로 레이어 내로 초점이 이동하여 레이어가 팝업됨을 알고 탐색하고 닫을 수 있도록 구현해야 함.
화면을 덮는 레이어를 닫은 경우, 초점이 레이어 팝업을 열기 이전의 요소로 자동적으로 복귀하지 않아 페이지를 처음부터 탐색해야 하며 이용이 불편함. 팝업된 레이어를 닫은 경우 자동적으로 레이어를 열도록 한 요소로 초점이 복귀하도록 구현해야 함.

24. 웹 애플리케이션 접근성 준수
웹 애플리케이션에 자체적인 접근성이 보장되지 없음. 사용자가 선택할 수 있는 대체 콘텐츠가 존재하지 않거나 적절하지 않게 제공한 경우
기본 입력서식이 아닌 별도의 요소로 구현된 기능을 스크린리더 사용자가 제작자가 의도한 요소로 인식할 수 없음. 코드를 수정하여 스크린리더 사용자가 기본 입력서식과 동일하게 인식하고 운용할 수 있도록 제공해야 함.
스크린리더 사용자는 가상키패드를 활성화 할 수 없음. 스크린리더 사용자도 가상키패드를 활성화여 사용할 수 있도록 수정해야 함.
스크린리더로는 해당 요소를 조작할 수 없음. 스크린리더로도 이용할 수 있도록 수정해야 함.
